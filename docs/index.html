How it works

Write YAML.
Get data.

1
Describe your target — define the site, CSS selectors, and what to do with the data in a YAML file.
2
Run the CLI — one command fetches, parses, transforms, validates, and saves.
3
Pick your output — JSON, CSV, SQLite, or MongoDB. All from the same directive.
directives/product.yaml 
site: https://books.toscrape.com/catalogue/a-light-in-the-attic_1000
use: beautifulsoup

scrape:
  title:
    - 'h1'
    - attr: text
  price:
    - 'p.price_color'
    - attr: text
  rating:
    - 'p.star-rating'
    - attr: class

transform:
  price:
    - strip
    - replace: {"£": ""}
    - float
  rating:
    - last    # "star-rating Three" → "Three"

validate:
  price:
    required: true
    type: float
    min: 0

Features

Everything you need,
nothing you don't

All configured from YAML. No code required for new scraping targets.

⚡

Two backends

BeautifulSoup for fast static pages. Playwright for JavaScript-rendered sites. Switch with one line in the YAML.

📄

Pagination

Follow "next page" links automatically. Define the selector once, Scrapit handles the rest up to max_pages.

🕷️

Spider mode

Discover and scrape all linked pages from an index. Set mode: spider and a link selector — done.

🔄

Transform pipeline

15+ declarative transforms: strip, regex, float, replace, slugify, template and more. All in YAML.

✅

Validation

Ensure data quality with per-field rules: required, type, min/max, pattern, in.

💾

4 storage backends

JSON, CSV (append mode), SQLite (zero-config), and MongoDB. Pick one per run with a CLI flag.

⚡

HTTP cache

File-based cache with configurable TTL. Stop hitting the site on every dev iteration.

🔔

Change detection

Diff results against previous runs. Fire a webhook automatically when content changes.

🔌

Hook system

Register callbacks for before_scrape, after_scrape, on_error, on_change. Extend without forking.

AI Integrations

Plug into any AI agent

Use Scrapit as a tool in LangChain, CrewAI, LangGraph, or LlamaIndex — your agent gets the ability to scrape the web on demand.

LangChain / CrewAI Tool

from scraper.integrations.langchain import ScrapitTool

tools = [ScrapitTool()]

# plug into any LangChain agent
agent = initialize_agent(
  tools=tools,
  llm=ChatOpenAI(model="gpt-4o"),
  agent=AgentType.OPENAI_FUNCTIONS,
)

LlamaIndex RAG

from scraper.integrations.llamaindex import ScrapitReader

reader = ScrapitReader()
docs = reader.load_data(url="https://example.com")

index = VectorStoreIndex.from_documents(docs)
engine = index.as_query_engine()
engine.query("Summarize the main points.")

Quick API No YAML needed

from scraper.integrations import scrape_url

# clean LLM-ready text, no config needed
text = scrape_url("https://example.com")

# or run a full directive
from scraper.integrations import scrape_directive
data = scrape_directive("wikipedia")
